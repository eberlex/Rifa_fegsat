# Como usar o site da Rifa (Rifa Fegsat)

Este guia explica o que colocar em cada caixa do formulário e como registrar uma venda.

## 1) Acessar o site
Abra no navegador o link do site onde o programa está rodando:
- https://rifa-fegsat.onrender.com/

Na página principal você verá o formulário **“Registrar Venda”**.

## 2) Preencher as caixas do formulário
Preencha todos os campos (todos são obrigatórios):

1. **Nome**
   - Coloque o nome completo da pessoa que está comprando a rifa.

2. **Telefone**
   - Coloque um telefone de contato (pode ser WhatsApp).

3. **Email**
   - Coloque o email do comprador.

4. **Número**
   - Coloque o **número da rifa** que o comprador está levando.
   - Exemplo: `12`, `34`, `101`.
   - O sistema impede que o mesmo número seja vendido duas vezes.

5. **Vendedor**
   - Coloque o nome de quem está realizando a venda (o vendedor/representante).
   - Exemplo: `João`, `Equipe Fegsat`, `Maria (vendedora)`.

## 3) Registrar a venda
Depois de preencher tudo, clique no botão **“Registrar”**.

### O que acontece quando você registra
- O sistema valida se nenhum campo ficou vazio.
- Ele verifica se o **Número** já não foi vendido antes.
- Se estiver tudo certo, ele salva a venda na planilha do Google Sheets.
- Em seguida, você é redirecionado para a página principal.

## 4) Conferir números vendidos
Para ver os números que já foram vendidos:
- Clique em **“Ver Números Vendidos”**.

Essa página lista os números vendidos até o momento.

## 5) Erros comuns
- **“Preencha todos os campos”**
  - Falta algum dado no formulário.
- **“O número X já foi vendido”**
  - Esse número já existe registrado; escolha outro número.

---
### Observação
Os dados das vendas ficam organizados na planilha (Google Sheets). O campo **Número** é usado para identificar quais números aparecem na tela **“Números Vendidos”**.
